Overview

What is
this project?

Fighter AI is an intelligent fighting game agent that plays against a human opponent in real time. Two neural networks operate in a chained pipeline every frame: a bidirectional LSTM reads the human's move history and predicts their next action, then a 1D CNN uses that prediction alongside the current game state to select the optimal counter-action.

Both models were first trained with supervised learning on 1 million+ frames of scripted bot self-play, then fine-tuned using REINFORCE reinforcement learning — two CNN agents playing against each other until both converge on strategic fighting behaviour beyond what any scripted bot could teach.

A physical Arduino-controlled servo rotates 5 times whenever the AI fighter is defeated, providing a tangible physical robot response to the simulation outcome.

Reflection

Results Discussion
& Future Work

✅

What worked well

Chaining LSTM and CNN into a single inference pipeline produced an AI that demonstrably adapts to player patterns. The ablation study confirmed the LSTM input meaningfully improves CNN counter-action selection. Class-weighted cross-entropy successfully prevented the LSTM from ignoring rare moves like Block and Uppercut. The temperature parameter gave fine-grained difficulty control with no additional training.

⚠️

Limitations

Training on CPU significantly increased iteration time — RL self-play in particular would benefit greatly from a GPU. The custom game environment, while fully functional, lacks the visual richness of a commercial fighting game which limits the demo's immediate impact. The LSTM window of 10 frames may be too short to detect slow-developing combo patterns.

🔭

Future work

Replacing the structured feature vector with raw pixel input would allow a true CNN vision pipeline (processing game frames directly). Extending the RL training with PPO instead of vanilla REINFORCE would improve sample efficiency. Integrating a Transformer-based move predictor as an ablation against the LSTM would complete the model comparison story.
